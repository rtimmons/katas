Installation
------------

To install, run this command in your terminal:

.. code-block:: console

   $ pip install katas


Usage
-----

.. code-block:: console

   $ katas [OPTIONS]

.. option:: --version

   Display the version and exit.

.. option:: --help

   Display a short usage message and exit.
